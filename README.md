# ⛵ ElytraScan — Fabric мод для Minecraft 1.20.1

Сканирует указанные блоки в загруженных чанках во время полёта на элитрах
и сохраняет все найденные координаты в текстовый файл.

---

## Функции

| Функция | Описание |
|---------|----------|
| **Сканирование чанков** | При полёте на элитрах автоматически проверяет все Y-уровни в каждом загруженном чанке — блоки под землёй не пропускаются |
| **GUI по Insert** | Полный интерфейс: список целей, пресеты, радиус, статистика |
| **Лог-файл** | Координаты сохраняются в `.minecraft/elytrascan_logs/<мир>_<дата>.txt` |
| **Умный сброс** | При смене мира/сервера — новый лог-файл, счётчики обнулятся |
| **HUD** | Статус в углу экрана + всплывающее уведомление при находке |
| **Пресеты** | Быстрое добавление: сундук, спавнер, руда, обломки и др. |

---

## Управление

- **Insert** — открыть/закрыть GUI
- **Delete** — удалить выбранный блок из списка (пока открыт GUI)
- **Enter** — добавить блок из поля ввода
- **Скролл мыши** — прокрутка списка

---

## Лог-файл

Путь: `.minecraft/elytrascan_logs/`

Формат строки:
```
[2024-03-15 14:22:01] [minecraft:overworld]  minecraft:chest          X=152    Y=63   Z=-88    (игрок: 150 / 120 / -90)
```

## Параметры конфига
