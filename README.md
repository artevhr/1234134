# ⛵ ElytraScan — Fabric мод для Minecraft 1.20.1

Сканирует указанные блоки в загруженных чанках во время полёта на элитрах
и сохраняет все найденные координаты в текстовый файл.

---

## Функции

| Функция | Описание |
|---------|----------|
| **Сканирование чанков** | При полёте на элитрах автоматически проверяет все Y-уровни в каждом загруженном чанке — блоки под землёй не пропускаются |
| **GUI по Insert** | Полный интерфейс: список целей, пресеты, радиус, статистика |
| **Лог-файл** | Координаты сохраняются в `.minecraft/elytrascan_logs/<мир>_<дата>.txt` |
| **Умный сброс** | При смене мира/сервера — новый лог-файл, счётчики обнулятся |
| **HUD** | Статус в углу экрана + всплывающее уведомление при находке |
| **Пресеты** | Быстрое добавление: сундук, спавнер, руда, обломки и др. |

---

## Управление

- **Insert** — открыть/закрыть GUI
- **Delete** — удалить выбранный блок из списка (пока открыт GUI)
- **Enter** — добавить блок из поля ввода
- **Скролл мыши** — прокрутка списка

---

## Установка

1. Установить **Fabric Loader 0.15.11+** для Minecraft 1.20.1
2. Установить **Fabric API** (скачать с modrinth.com/mod/fabric-api)
3. Скопировать `elytrascan-1.0.0.jar` в папку `.minecraft/mods/`
4. Запустить игру с профилем Fabric

---

## Сборка из исходников (нужен Java 17+)

```bash
# Windows:
gradlew build

# Linux / macOS:
chmod +x gradlew
./gradlew build
```

Готовый JAR появится в: `build/libs/elytrascan-1.0.0.jar`

> Первый запуск скачает Gradle (~130 МБ) и зависимости (~400 МБ).
> Нужен интернет и ~600 МБ места.

### Если нет gradlew:
```bash
gradle wrapper --gradle-version=8.3
./gradlew build
```

## Лог-файл

Путь: `.minecraft/elytrascan_logs/`

Формат строки:
```
[2024-03-15 14:22:01] [minecraft:overworld]  minecraft:chest          X=152    Y=63   Z=-88    (игрок: 150 / 120 / -90)
```

---

## Параметры конфига

`.minecraft/config/elytrascan.json`

```json
{
  "scanEnabled": false,
  "targetBlocks": ["minecraft:chest", "minecraft:spawner"],
  "scanRadius": 3,
  "onlyWhenElytra": true
}
```

| Поле | Описание | Диапазон |
|------|----------|----------|
| `scanEnabled` | Включено ли сканирование | true/false |
| `targetBlocks` | Список блоков для поиска | ID блоков |
| `scanRadius` | Радиус в чанках | 1–8 (16–128 блоков) |
| `onlyWhenElytra` | Сканировать только при полёте на элитрах | true/false |

---

## Важно

- Мод **только клиентский** — не нужен на сервере
- Сканируются только **загруженные** чанки (в радиусе прогрузки)
- Одни и те же координаты **не дублируются** в логе за сессию
- При нажатии "↺ Сбросить карту" — забываются все найденные позиции и запускается новый лог
